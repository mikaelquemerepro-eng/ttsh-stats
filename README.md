# Statistiques TTSH

Site de statistiques pour les matchs de tennis de table du TTSH.

## Accès au site

Ce site est hébergé sur GitHub Pages et affiche les résultats et statistiques des matchs.

## Mise à jour

Pour mettre à jour les statistiques :
1. Copier les nouveaux fichiers JSON dans le dossier approprié
2. Commit et push les changements
3. Le site sera automatiquement mis à jour

## Structure

- `index.html` : Page principale avec l'interface
- `styles.css` : Styles du site
- `script.js` : Logique JavaScript
- `statistiques.json` : Données statistiques globales
- `J3_20251012/` : Résultats de la journée 3
- `J4_20251116/` : Résultats de la journée 4
